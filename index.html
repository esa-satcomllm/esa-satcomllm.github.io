What the project delivers

Clear outcomes that you can use in work and research.

SatcomLLM

An open model adapted on satcom material, tuned for tasks like link margin checks, protocol questions and quick design notes.

SCEVA

A web app and API that serve the model, with retrieval for source backed answers and tools for expert review.

Dataset

A curated corpus from public satcom sources, cleaned and tagged for future research and community use.

How to ship

  1. Save this file as index.html in your repo.
  2. Push to the main branch or your Pages branch.
  3. Turn on Pages in repo settings and choose your source.
  4. Edit the title, text, and colors to match your brand.
